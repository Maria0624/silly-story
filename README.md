# 🎭 Silly Story Generator

A fun JavaScript project that generates random silly stories using arrays, functions, and DOM manipulation.

---

## 📌 About the Project

This app creates random humorous stories when the user clicks a button. It uses a fixed story template and fills in random values from predefined arrays. Users can also enter a custom name to personalize the story.

Each click generates a new and unique story.

---

## 🚀 Features

- Random story generation
- Custom name input option
- Dynamic text replacement
- Fully interactive button click
- New story every time

---

## 🧠 Concepts Used

- JavaScript Arrays
- Functions
- Random number generation (`Math.random`)
- DOM Manipulation
- Event Listeners
- String replacement
